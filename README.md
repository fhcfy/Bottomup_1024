# Bottomup_1024

검색어를 입력하면 해당 검색어와 관련된 단어들의 결과를 보여주고, 워드 클라우드를 만들어 저장할 수 있도록 하는 기능이 있는 안드로이드 어플리케이션

### 사용
 - android, android studio
 - java, AsyncTask, Jsoup
